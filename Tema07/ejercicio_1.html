Tema 7 - Funciones

Ejercicios prácticos para trabajar diferentes formas y patrones de funciones en JavaScript.

Ejercicio 1 - Función que devuelve valor (raíz cuadrada)

Implementa y prueba raizCuadrada(numero) que devuelva la raíz cuadrada.

Ejercicio 2 - Procedimiento (alerta)

Usa la función alerta(mensaje) para mostrar un mensaje. (Ejemplo del tema)

Nota: se mostrará un alert en el navegador.

Ejercicio 3 - Constructor Function

Crear una función usando new Function('a', 'b', 'return a + b;') y probarla.

Ejercicio 4 - Hoisting con var

Reproduce el ejemplo de hoisting con var: llama a la función antes de declarar la variable y observa el resultado.

Ejercicio 5 - Función autoinvocada (IIFE)

Crear una IIFE que muestre un mensaje en consola y capture su valor de retorno.

Ejercicio 6 - Función anónima

Define const dividir = function(a,b){...} y úsala para dividir dos números.

Ejercicio 7 - Función flecha

Crear const multiplicar = (a,b) => a * b; y probarla.

Ejercicio 8 - Parámetros por defecto

Usa function saludar(nombre = 'Invitado'){...} y prueba con/ sin argumento.

Ejercicio 9 - Funciones anidadas

Implementa externa() con una interna() que acceda a la variable de la externa.

Ejercicio 10 - Métodos nativos

Usa métodos nativos vistos: length, toUpperCase, trim, indexOf, Math.random(), Date.now().
